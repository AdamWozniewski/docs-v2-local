* [許可リスト用のAuth0 IPアドレス](/docs/ja-jp/secure/security-guidance/data-security/allowlist) - 環境がファイアウォールの背後にある場合、カスタムデータベース接続、アクション、フック、ルールが適切に機能するように、特定のAuth0 IPアドレスを許可リストに追加する必要があります。
* [ユーザー属性をDenyList（拒否リスト）に追加する](/docs/ja-jp/secure/security-guidance/data-security/denylist) - プライバシー上の理由でAuth0データベースに保存すべきでないユーザーフィールドがある場合、そのユーザー属性を拒否リストに追加できます。
* [ユーザーデータの保存](/docs/ja-jp/secure/security-guidance/data-security/user-data-storage) - Auth0データストアと外部データベースを使用してユーザーデータを保存する際の違いを説明します。
* [トークンの保管](/docs/ja-jp/secure/security-guidance/data-security/token-storage) - トークンベースの認証で使用するトークンをどこにどのように保存するかを説明します。
