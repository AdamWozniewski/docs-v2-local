Connect your AI agents to Hugging Face for model access, dataset management, and machine learning workflows.
It is designed for applications that need to retrieve, analyze, or automate work with state-of-the-art AI models and datasets.

When used with Auth0's [Token Vault](https://auth0-genai.mintlify.app/intro/token-vault), this connection can enable use cases such as:
- Access model information: retrieve model details, performance metrics, and usage statistics from Hugging Face.
- Support AI-powered workflows: integrate Hugging Face pipelines into your applications to automate machine learning tasks.
- Integrate with creative processes: connect Hugging Face models with collaboration tools, chat systems, or analysis pipelines to enable real-time insights and decision-making.

## Connect Hugging Face to Auth0

<IntegrationInfoBlock providerName="Hugging Face" />

<Steps>
  <Step title="Hugging Face setup">
    1. Visit the [Hugging Face Developer account](https://huggingface.co/) page and sign up.
    2. Navigate to the connected apps section under settings.
    3. Create a new OAuth application in the Hugging Face Developer Console.
    4. Complete the application form.
    5. Configure scopes:
       - Add the necessary scopes for your application.
    6. Configure OAuth redirect URL:
       - Redirect URL: `https://YOUR_AUTH0_DOMAIN/login/callback`
       <FindDomainInfoBlock />
    7. Click Create Application.
    8. After creation you'll be shown the client ID and client secret.
    9. Copy the Client ID and Client Secret.

    To learn more about setting up Hugging Face, read the [Developer guide for Hugging Face documentation](https://huggingface.co/docs/hub/oauth).
  </Step>
  <Step title="Auth0 setup">
    <Auth0SetupBlock providerName="Hugging Face"  />
  </Step>
</Steps>
